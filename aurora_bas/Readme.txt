Library install for default BAS installation path:
Create a folder AuroraLibrary by path:
C:\Users\UserName\AppData\Roaming\BrowserAutomationStudio\apps\21.6.2\modules\, where 21.6.2 - your BAS version
Result: C:\Users\UserName\AppData\Roaming\BrowserAutomationStudio\apps\21.6.2\modules\AuroraLibrary
Download and repack to AuroraLibrary folder all files from https://github.com/Pragmatikc/Aurora/tree/current
Restart your BAS project

Установка библиотеки для стандартного пути установки БАС:
Создайте папку AuroraLibrary по адресу:
C:\Users\UserName\AppData\Roaming\BrowserAutomationStudio\apps\21.6.2\modules\, где 21.6.2 - ваша версия БАС
Результат: C:\Users\UserName\AppData\Roaming\BrowserAutomationStudio\apps\21.6.2\modules\AuroraLibrary
Скачайте и распакуйте в папку AuroraLibrary все файлы из ветки репозитория https://github.com/Pragmatikc/Aurora/tree/current
Перезапустите ваш проект
